# Контекст команды РЭМД

## Иерархия и роли

### Руководители проектов
- **Ильназ** - РП РЭМД (руководитель проекта РЭМД)
  - Приоритет задач: HIGH/CRITICAL
  - Роль: ключевые решения по проекту РЭМД

- **Настя** - РП РЭМД от ЦМС (руководитель проекта от компании ЦМС)
  - Приоритет задач: HIGH/CRITICAL
  - Роль: координация со стороны ЦМС, инфраструктура

- **Максим** - РП КУ ФЭР (руководитель проекта КУ ФЭР)
  - Приоритет задач: MEDIUM/HIGH
  - Роль: смежный проект, координация по КУ ФЭР

### Непосредственное руководство
- **Саида** - начальник отдела аналитики
  - Приоритет задач: HIGH (непосредственный начальник)
  - Роль: управление отделом, постановка задач аналитикам

## Правила приоритизации

1. **CRITICAL/HIGH приоритет** - задачи от:
   - Ильназа (РП РЭМД)
   - Саиды (непосредственный начальник)
   - Насти (РП от ЦМС)
   - Критичные системы: ФРМО, СЭМД, ФЛК

2. **MEDIUM/HIGH приоритет** - задачи от:
   - Максима (РП смежного проекта)
   - Задачи по ЕГИСЗ
   - Координационные задачи

3. **MEDIUM/LOW приоритет** - задачи:
   - Улучшения и автоматизация
   - Организационные задачи
   - Бэклог

## Ключевые системы проекта РЭМД

- **РЭМД** - Российская Электронная Медицинская Документация
- **СЭМД** - Структурированные Электронные Медицинские Документы (критичная система)
- **ФЛК** - Формально-Логический Контроль (валидация медицинских данных)
- **ГИСЗ/ЕГИСЗ** - Государственная Информационная Система Здравоохранения
- **МИС** - Медицинские Информационные Системы
- **ФРМО** - Федеральный Регистр Медицинских Организаций
- **КУ ФЭР** - Контроль Учета ФЭР (смежный проект)
