Жди от пользователя список задач. Он может быть в любом формате:
- Нумерованный список (1. 2. 3.)
- Просто строки
- С Jira ссылками (EGISZREMD-123, AN-456)
- С упоминаниями людей (от Максима, задача Саиды)
- Со словами срочности (СРОЧНО, критично, блокер)

Для каждой задачи:
1. Извлеки Jira ссылки
2. Найди упоминания людей
3. Определи слова срочности
4. Классифицируй по типу: Анализ/Документация/Разработка/Координация/Баг
5. Оцени сложность: низкая/средняя/высокая
6. Определи приоритет: 1-5 (учитывай слова срочности, связь с ФЛК/СЭМД/ГИСЗ)
7. Оцени время в часах

Прочитай data/tasks.json, добавь новые задачи, сохрани обратно.

Покажи что добавлено в удобном формате.
