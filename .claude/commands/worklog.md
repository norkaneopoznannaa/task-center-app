---
description: Добавить запись о выполненной работе
---

Пользователь укажет:
- Задачу (по названию, части ID или Jira ключу)
- Время работы (в минутах или часах, например "2ч" или "30м")
- Опционально: описание что сделано

Действия:
1. Найди задачу в `data/tasks.json` по указанным критериям
2. Добавь worklog запись в массив `worklogs` задачи:
   ```json
   {
     "date": "YYYY-MM-DD",
     "minutes": число,
     "description": "описание",
     "synced_to_jira": false
   }
   ```
3. Обнови `time_tracking.total_minutes` задачи
4. Сохрани изменения в `data/tasks.json`
5. Покажи результат

Примеры использования:
- "EGISZREMD-15282 2ч анализ требований"
- "задача про ФЛК 45м"
- "первая задача 1.5ч"

Если задача связана с Jira - предложи синхронизировать worklog.
