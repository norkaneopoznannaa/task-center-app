---
description: Сформировать отчет о работе за период
---

Пользователь может указать период:
- "сегодня" / "today"
- "неделя" / "week"
- "месяц" / "month"
- Конкретные даты "01.01.2026 - 15.01.2026"

Действия:
1. Прочитай `data/tasks.json`
2. Собери все worklogs за указанный период
3. Сгруппируй по:
   - Категориям (РЭМД, КУ ФЭР, общие, авто)
   - Проектам Jira
   - Задачам

4. Сформируй отчет:

```
## Отчет за [период]

### Общая статистика
- Всего времени: X ч Y мин
- Задач выполнено: N
- Задач в работе: M

### По категориям
| Категория | Время | % |
|-----------|-------|---|
| РЭМД      | Xч    | Y |
| КУ ФЭР    | Xч    | Y |

### Детализация
1. [Задача] - Xч
   - Jira: EGISZREMD-123
   - Worklogs: [даты и описания]
```

5. Если есть несинхронизированные worklogs в Jira - предупреди.
